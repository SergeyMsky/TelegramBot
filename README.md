```
git clone https://github.com/SergeyMsky/TelegramBot.git
```

Create a bot via [@BotFather](https://telegram.me/BotFather) and grab a token.

```javascript
const bot = new TelegramBot(YOUR_TOKEN, { polling: true })
```

Install all dependencies:

`npm install`

Launch app:

`npm run dev`
